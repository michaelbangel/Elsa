# elsa
Layered financial representation analytics framework designed to identify patterns of financial reporting variance and inaccuracies during negotiations with stakeholders and Financial Risk Analytics Framework

This project analyzes financial variance, settlement outcomes, operational risk patterns, and governance indicators using R.

It includes:
- Variance detection between reported and verified financials
- Settlement intelligence scoring
- Litigation exposure indicators
- Fraud risk indicators
- Financial Integrity Index (FII)
- Post-settlement risk detection
